# sec-learning

セキュリティ資格試験の対策用に作成した、ブラウザで動作する模擬試験アプリです。

## ファイル一覧

| ファイル | 内容 |
|---|---|
| `index.html` | 模擬試験アプリ（125問 × 5セット） |

## 機能

- **625問** （125問 × 5セット）をブラウザ単体で実行
- 選択肢は毎回**ランダム配置**（暗記防止）
- 各選択肢の**文字数を均一化**（正解だけが長い手がかりを排除）
- **カウントダウンタイマー**（4時間、一時停止・リセット対応）
- 解答後に**解説**を即時表示
- **正解率・進捗**をリアルタイム表示
- 解答状況を**LocalStorage に自動保存**（ページを閉じても続きから再開可能）
- **復習モード**：不正解問題のみ・正解問題のみをフィルタリング
- **ドメイン別スコア**：分野ごとの得点を結果画面に表示
- **印刷 / PDF 出力**：不正解一覧をまとめてプリント

## 使い方

1. `index.html` をブラウザで開く
2. セット（Set 1〜5）を選択する
3. タイマーの「開始」ボタンを押して試験を開始する
4. 各問題の選択肢をクリックして解答する
5. 「結果を見る」ボタンで得点と分野別分析を確認する

> ネット接続なしでも動作しますが、Google Fonts の読み込みにはインターネット接続が必要です。

## 技術スタック

- **HTML / CSS / Vanilla JavaScript** のみ（依存ライブラリなし）
- LocalStorage を使った状態管理
- レスポンシブデザイン対応
