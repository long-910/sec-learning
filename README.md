# sec-learning

セキュリティ学習用に作成した、ブラウザで動作する学習アプリです。

## ファイル一覧

| ファイル | 内容 |
|---|---|
| `index.html` | 学習アプリ本体（UIのみ） |
| `questions.js` | 問題データ（125問 × 10セット） |

## 機能

- **1,250問** （125問 × 10セット）をブラウザ単体で実行
- 選択肢は毎回**ランダム配置**（暗記防止）
- 各選択肢の**文字数を均一化**（正解だけが長い手がかりを排除）
- **カウントダウンタイマー**（4時間、一時停止・リセット対応）
- 解答後に**解説**を即時表示
- **正解率・進捗**をリアルタイム表示
- 解答状況を**LocalStorage に自動保存**（ページを閉じても続きから再開可能）
- **復習モード**：不正解問題のみ・全問をフィルタリングして復習
- **ドメイン別スコア**：分野ごとの得点を結果画面に表示
- **印刷 / PDF 出力**：不正解一覧をまとめてプリント
- **セットボタンに進捗表示**：途中は `セット 1 · 45/125`、完了は `セット 1 ✓ 88%` を表示
- **全体ダッシュボード**：10セット横断の解答済数・正解数・正答率と、ドメイン別正答率（弱点順）を一覧表示

## 使い方

1. `index.html` をブラウザで開く（`questions.js` と同じフォルダに置くこと）
2. パスワードを入力してログインする
3. セット（セット 1〜10）を選択する
4. タイマーの「開始」ボタンを押すか、最初の問題に解答すると計測が始まる
5. 「結果を見る」ボタンで得点と分野別分析を確認する
6. 「全体ダッシュボード」ボタンで全セットの進捗をまとめて確認できる

## 問題セットの追加方法

`questions.js` の `ALL_SETS` 配列に要素を追加するだけです。

```js
// questions.js
const ALL_SETS = [
  [ /* Set 1: 既存の125問 */ ],
  // ...
  [
    {
      "d": "ドメイン名（分野）",
      "q": "問題文",
      "o": ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
      "a": 0,  // 正解のインデックス（0始まり）
      "e": "解説文"
    },
    // ... 続けて問題を追加
  ]
];
```

> **ヒント:** 各セットの問題数は任意です。追加後、ブラウザをリロードすると新しいセットがボタンに表示されます。

> ネット接続なしでも動作しますが、Google Fonts の読み込みにはインターネット接続が必要です。

## 技術スタック

- **HTML / CSS / Vanilla JavaScript** のみ（依存ライブラリなし）
- LocalStorage を使った状態管理
- レスポンシブデザイン対応
